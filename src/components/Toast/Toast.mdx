# Toast

사용자 인터페이스에서 잠깐 나타났다가 사라지는 알림 메시지를 표시하는 컴포넌트

## Example

```jsx
import { ToastProvider } from 'ventileco-ui'

ReactDOM.createRoot(document.getElementById('root')!).render(
  <React.StrictMode>
    <ToastProvider>
      {({ toasts }) => (
        <React.Fragment>
          <App />
          <ToastProvider.List>
            {toasts.map((toast) => (
              <ToastProvider.Wrapper key={toast.id} toastId={toast.id}>
                // Toast로 사용하고 싶은 UI를 작성하세요.
              </ToastProvider.Wrapper>
            ))}
          </ToastProvider.List>
        </React.Fragment>
      )}
    </ToastProvider>
  </React.StrictMode>,
)
```

```jsx
import { useToast } from 'ventileco-ui'

function App() {
  const addToast = useToast()

  return (
    <div>
      <button onClick={() => addToast.success('성공 메시지')}>버튼</button>
      <button onClick={() => addToast.error('실패 메시지')}>버튼</button>
      <button onClick={() => addToast.info('인포 메시지')}>버튼</button>
    </div>
  )
}

export default App
```
