# Toast

사용자 인터페이스에서 잠깐 나타났다가 사라지는 알림 메시지를 표시하는 컴포넌트

## Example

```jsx
import { ToastProvider } from 'ventileco-ui'

ReactDOM.createRoot(document.getElementById('root')!).render(
  <React.StrictMode>
    <ToastProvider>
      {({ toasts }) => (
        <React.Fragment>
          <App />
          <ToastProvider.List>
            {toasts.map((toast) => (
              <ToastProvider.Wrapper key={toast.id} id={toast.id}>
                // Toast로 사용하고 싶은 UI를 작성하세요.
              </ToastProvider.Wrapper>
            ))}
          </ToastProvider.List>
        </React.Fragment>
      )}
    </ToastProvider>
  </React.StrictMode>,
)
```

```jsx
import { useToast } from 'ventileco-ui'

function App() {
  const addToast = useToast()

  return (
    <div>
      <button onClick={() => addToast.success('성공 메시지')}>버튼</button>
      <button onClick={() => addToast.error('실패 메시지')}>버튼</button>
      <button onClick={() => addToast.info('인포 메시지')}>버튼</button>
    </div>
  )
}

export default App
```

## With Framer Motion

```jsx
import { ToastProvider } from 'ventileco-ui'
import { motion } from 'framer-motion'

ReactDOM.createRoot(document.getElementById('root')!).render(
  <React.StrictMode>
    <ToastProvider>
      {({ toasts }) => (
        <React.Fragment>
          <App />
          <ToastProvider.List>
            {toasts.map((toast) => (
              <ToastProvider.Wrapper key={toast.id} id={toast.id} motion={motion}>
                <div className="flex h-[80px] w-[320px] items-center rounded-full border px-4 py-2">
                  <div
                    className={`mr-2 size-5 shrink-0 ${toast.type === 'success' ? 'bg-success' : toast.type === 'error' ? 'bg-error' : 'bg-info'}`}
                  />
                  <div>
                    <p
                      className={`mr-2 text-lg font-semibold ${toast.type === 'success' ? 'text-success' : toast.type === 'error' ? 'text-error' : 'text-info'}`}
                    >
                      {toast.type}
                    </p>
                    <p className="text-sm">{toast.message}</p>
                  </div>
                </div>
              </ToastProvider.Wrapper>
            ))}
          </ToastProvider.List>
        </React.Fragment>
      )}
    </ToastProvider>
  </React.StrictMode>,
)
```
