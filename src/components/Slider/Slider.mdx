# Slider

클릭이나 드래그를 사용한 상호작용으로 수평으로 이동 가능한 바를 제공하는 인터페이스 요소

## Example

```jsx
import Slider from './ventileco-ui

function Example() {
  return (
    <Slider gap={} step={}>
      <Slider.PrevButton className=""></Slider.PrevButton>
      <Slider.Content></Slider.Content>
      <Slider.NextButton className=""></Slider.NextButton>
    </Slider>
  )
}
```
