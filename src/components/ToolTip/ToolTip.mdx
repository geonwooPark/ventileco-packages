# ToolTip

사용자 인터페이스 요소 위에 마우스를 올렸을 때 나타나는 작은 팝업 창으로, 추가 정보나 설명을 제공하여 사용자 경험을 향상시키는 역할을 하는 인터페이스 요소

## Example

```jsx
import ToolTip from './ventileco-ui

function Example() {
  return (
    <ToolTip direction='top'>
      <ToolTip.Trigger>
        <div className="rounded-md bg-gray-800 px-2 py-1 text-white">
          Trigger
        </div>
      </ToolTip.Trigger>
      <ToolTip.Content>
        <div className="rounded-md bg-black px-3 py-5 text-white shadow-md">
          Content
        </div>
      </ToolTip.Content>
    </ToolTip>
  )
}
```
