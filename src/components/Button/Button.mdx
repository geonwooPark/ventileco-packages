# Button

사용자의 클릭 이벤트를 처리하여 다양한 동작을 수행하는 인터페이스 요소

## Example

```jsx
import Button from './ventileco-ui

function Example() {
  return (
      <Button type={} className={}>
        <Button.Icon></Button.Icon>
        <Button.Label></Button.Label>
      </Button>
  )
}
```
