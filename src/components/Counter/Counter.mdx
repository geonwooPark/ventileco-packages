# Counter

사용자가 클릭 이벤트를 통해 숫자 값을 증가 또는 감소시킬 수 있는 UI 요소

## Example

```jsx
import { Counter } from 'ventileco-ui'

function Example() {
  const [value, setValue] = useState(3)

  return (
    <Counter value={value} setValue={setValue} maximum={10} minimum={0}>
      <div className="flex items-center gap-2">
        <Counter.Number className="h-8 w-20 rounded-md border border-gray-700 text-center text-2xl" />
        <div className="flex flex-col">
          <Counter.Up>
            <svg
              xmlns="http://www.w3.org/2000/svg"
              fill="none"
              viewBox="0 0 24 24"
              strokeWidth="1.5"
              stroke="currentColor"
              className="size-6"
            >
              <path
                strokeLinecap="round"
                strokeLinejoin="round"
                d="m4.5 15.75 7.5-7.5 7.5 7.5"
              />
            </svg>
          </Counter.Up>
          <Counter.Down>
            <svg
              xmlns="http://www.w3.org/2000/svg"
              fill="none"
              viewBox="0 0 24 24"
              strokeWidth="1.5"
              stroke="currentColor"
              className="size-6"
            >
              <path
                strokeLinecap="round"
                strokeLinejoin="round"
                d="m19.5 8.25-7.5 7.5-7.5-7.5"
              />
            </svg>
          </Counter.Down>
        </div>
      </div>
    </Counter>
  )
}
```
